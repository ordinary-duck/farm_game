========================================
  Button Images Folder
========================================

Place your custom button images here!

Recommended Sizes:
------------------
- Regular buttons: 300 x 60 pixels
- Arrow buttons: 60 x 60 pixels
- Start button: 300 x 60 pixels

Supported Formats:
------------------
- PNG (recommended, supports transparency)
- JPG / JPEG
- BMP
- GIF

Example Button Names:
---------------------
- btn_start.png        - Start game button
- btn_prev.png         - Previous/Left arrow button
- btn_next.png         - Next/Right arrow button
- btn_default.png      - Default button style

How to Use:
-----------
1. Place your button image files in this folder
2. Open config.py
3. Set the button image paths:

   BUTTON_START_IMAGE = r'assets\buttons\btn_start.png'
   BUTTON_PREV_IMAGE = r'assets\buttons\btn_prev.png'
   BUTTON_NEXT_IMAGE = r'assets\buttons\btn_next.png'

4. Run the game!

Tips:
-----
- Use PNG format for best results
- Keep file sizes reasonable (< 500KB each)
- Use consistent style across all buttons
- Test different designs to see what looks best

Free Resources:
---------------
- OpenGameArt.org
- itch.io (Free Assets)
- Kenney.nl
- Create your own with GIMP, Photoshop, or Canva
