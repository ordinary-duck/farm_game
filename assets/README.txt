===============================================
游戏资源文件夹 / Game Assets Folder
===============================================

这个文件夹用于存放游戏的图片、音效等资源文件。

如何添加自定义登录背景：
--------------------------

1. 将你的背景图片复制到这个文件夹
   
2. 打开 farm_game/config.py 文件

3. 找到 LOGIN_BG_IMAGE 配置行，修改为：
   LOGIN_BG_IMAGE = r'assets\你的图片名.jpg'
   
   例如：
   LOGIN_BG_IMAGE = r'assets\login_bg.jpg'
   或：
   LOGIN_BG_IMAGE = r'D:\桌面\爆炸\farm_game\assets\background.png'

4. 重新运行游戏

支持的图片格式：
--------------------------
- JPG / JPEG
- PNG
- BMP
- GIF
- 其他常见格式

建议的图片尺寸：
--------------------------
- 960x480 像素（与当前窗口大小匹配）
- 或更大（会自动缩放）

提示：
--------------------------
- 图片会自动缩放到窗口大小
- 建议使用横版图片
- 文件名可以包含中文
- 路径前要加 r，例如：r'assets\图片.jpg'
