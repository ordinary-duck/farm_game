# 中文字体显示修复说明

## 问题
游戏中的中文字符（如玩家名字"阿农"、"小叶"、"老王"）显示为方块（口口）。

## 原因
Pygame 默认字体不支持中文字符。

## 解决方案
已自动添加 Windows 系统中文字体支持。

## 修改的文件
1. **config.py** - 添加了字体路径检测
2. **login_scene.py** - 使用中文字体
3. **farm_scene.py** - 使用中文字体
4. **inventory_ui.py** - 使用中文字体

## 支持的字体（按优先级）
1. 微软雅黑 (msyh.ttc) - 推荐
2. 黑体 (simhei.ttf)
3. 宋体 (simsun.ttc)
4. 楷体 (simkai.ttf)

## 测试
现在重新运行游戏，中文应该正常显示！

```bash
python main.py
```

或双击 `start.bat`

## 预期效果

- ? 农场名字正常显示
- ? 物品名字正常显示
- ? 所有UI文字正常显示

## 如果还是显示方块
1. 检查 C:\Windows\Fonts 文件夹中是否有中文字体
2. 查看控制台输出，应该显示 "Using Chinese font: ..."
3. 确保字体文件未损坏
